# Cephalo

## Development setup

[Install Node.js](https://nodejs.org/en/download)

Install yarn

```
npm install --global yarn
```

Install dependencies

```
npm install
```

For local development

```
npm run dev
```

Formatting

```
yarn format
```

## Deploy to GitHub pages

```
npm run build
npm run deploy
```
